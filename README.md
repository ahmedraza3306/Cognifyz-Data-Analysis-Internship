# Cognifyz Data Analysis Internship

## Overview
This repository contains all tasks completed during the Cognifyz Data Analysis Internship. The project focuses on analyzing the Zomato Restaurant Dataset using Python, SQL, and data visualization techniques to extract meaningful business insights.

- Matplotlib

- MySQL 8.0
- Git & GitHub
